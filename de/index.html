Der Geviertstrich (—) ist ein vielseitiges Satzzeichen, das für Betonung, Unterbrechungen oder zur Hervorhebung eines Satzteils verwendet wird. So geben Sie es auf verschiedenen Plattformen ein:

Mac

Option + Umschalt + -

Windows

Alt + 0151 (auf dem Ziffernblock)

Linux

Strg + Umschalt + U, dann 2014, drücken Sie Enter

Microsoft Word

Tippen Sie --- (drei Bindestriche) → automatischer Ersatz

Google Docs

Tippen Sie --- (drei Bindestriche) → automatischer Ersatz

Notion

Tippen Sie --- oder fügen Sie es ein

Telegram

Kopieren und einfügen von woanders — keine automatische Korrektur

WhatsApp

Kopieren und einfügen von woanders — keine automatische Korrektur

Android

Halten Sie die - Taste gedrückt → wählen Sie —

Markdown

Schreiben Sie ---

&mdash;

Über den Geviertstrich

Der Geviertstrich ist nach seiner Breite benannt—traditionell ist er so breit wie der Großbuchstabe M. Er ist länger als der Bindestrich (-) und der Halbgeviertstrich (–). Gevierstriche können Kommas, Semikolons, Doppelpunkte und Klammern ersetzen, um zusätzliche Betonung, Unterbrechung oder einen abrupten Gedankenwechsel anzuzeigen.

Denken Sie daran—entgegen dem, was das Internet sagt—nicht nur KI-Autoren lieben es, Gevierstriche übermäßig zu verwenden. Menschliche Autoren haben sie—ziemlich meisterhaft—seit Jahrhunderten überstrapaziert!
