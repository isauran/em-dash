Le tiret cadratin (—) est un signe de ponctuation polyvalent utilisé pour l'emphase, les interruptions ou pour mettre en évidence une partie d'une phrase. Voici comment le taper sur différentes plateformes :

Mac

Option + Maj + -

Windows

Alt + 0151 (sur le pavé numérique)

Linux

Ctrl + Maj + U, puis 2014, appuyez sur Entrée

Microsoft Word

Tapez --- (trois traits d'union) → remplacement automatique

Google Docs

Tapez --- (trois traits d'union) → remplacement automatique

Notion

Tapez --- ou collez-le

Telegram

Copiez-collez depuis ailleurs — pas de correction automatique

WhatsApp

Copiez-collez depuis ailleurs — pas de correction automatique

Android

Maintenez la touche - → sélectionnez —

iOS

Maintenez la touche - → sélectionnez —

Markdown

Écrivez ---

HTML

&mdash;

À propos du Tiret Cadratin

Le tiret cadratin tient son nom de sa largeur—traditionnellement, il est aussi large que la lettre majuscule M. Il est plus long que le trait d'union (-) et le tiret demi-cadratin (–). Les tirets cadratins peuvent remplacer les virgules, les points-virgules, les deux-points et les parenthèses pour indiquer un accent supplémentaire, une interruption ou un changement brusque de pensée.

Souvenez-vous—contrairement à ce que dit Internet—ce ne sont pas seulement les rédacteurs IA qui adorent utiliser les tirets cadratins de manière excessive. Les écrivains humains en abusent—avec beaucoup de talent—depuis des siècles !
