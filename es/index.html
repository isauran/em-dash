La raya (—) es un signo de puntuación versátil que se usa para enfatizar, interrupciones o para destacar parte de una oración. Así es como se escribe en varias plataformas:

Mac

Option + Shift + -

Windows

Alt + 0151 (en el teclado numérico)

Linux

Ctrl + Shift + U, luego 2014, presiona Enter

Microsoft Word

Escribe --- (tres guiones) → reemplazo automático

Google Docs

Escribe --- (tres guiones) → reemplazo automático

Notion

Escribe --- o pégalo

Telegram

Copia y pega desde otro lugar — no hay corrección automática

WhatsApp

Copia y pega desde otro lugar — no hay corrección automática

Android

Mantén presionada la tecla - → selecciona —

iOS

Mantén presionada la tecla - → selecciona —

Markdown

Escribe ---

HTML

&mdash;

Sobre la Raya

La raya recibe su nombre por su anchura—tradicionalmente, es tan ancha como la letra mayúscula M. Es más larga que tanto el guion (-) como la raya n (–). Las rayas pueden reemplazar comas, puntos y comas, dos puntos y paréntesis para indicar énfasis adicional, interrupción o un cambio abrupto de pensamiento.

Recuerda—a pesar de lo que dice Internet—no solo los escritores de IA aman usar las rayas en exceso. Los escritores humanos también han abusado de ellas—con bastante maestría—durante siglos!
